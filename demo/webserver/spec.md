# Webserver Demo Spec

## Claim

Agents can build a complete, working web stack — native TCP server, static file serving, WASM fractal renderer, browser client — authored entirely in LLVM IR, without frameworks, high-level languages, or hand-written build configuration. The browser-facing JS shim is under 50 lines and provides only raw DOM syscalls; all application logic is in the WASM module compiled from IR.

This demo reaches **L1 conformance**: PCF metadata is attached to all gated functions and `verify.sh` / `link-gate.sh` check structural completeness and consistency. Proof discharge is not solver-backed — that is the role of the storage demo.

## Goal

Demonstrate a pure-LLVM web stack for Alien Stack:
- HTTP server in LLVM IR.
- Fractal renderer in WASM generated from LLVM IR.
- Minimal browser page where JS only loads WASM and blits pixels.

## Source of Truth

- Server IR: `demo/webserver/server.ll`
- Fractal IR: `demo/webserver/fractal.ll`
- Web client: `demo/webserver/public/index.html`
- Build pipeline: `demo/webserver/build.sh`
- Contract gate: `demo/webserver/verify.sh`
- Link gate: `demo/webserver/link-gate.sh`
- Artifact seal: `demo/webserver/seal-artifacts.sh`

## Runtime Behavior

Routes:
- `GET /` -> `public/index.html`
- `GET /fractal.wasm` -> `public/fractal.wasm`
- Any other path -> static 404 response

The server preloads and prebuilds full HTTP responses at startup to minimize hot-path work.

## Contract Requirements

Required gated functions in IR:
- `build_response`
- `read_file`
- `get_content_type`
- `check_invariants`
- `load_assets`
- `handle_client`
- `main`
- `generate_fractal`
- `get_buffer`
- `get_buffer_size`
- `free_buffer`

Each must include:
- `pcf.schema`
- `pcf.toolchain`
- `pcf.pre`
- `pcf.post`
- `pcf.proof`
- `pcf.effects`
- `pcf.bind`

Gate behavior:
- `verify.sh` fails closed on missing/invalid metadata.
- `link-gate.sh` fails closed on unresolved gated edge requirements.

## Build and Run

```bash
cd demo/webserver
./build.sh
./run.sh
```

Open `http://localhost:9090`.
