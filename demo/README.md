# Demos

- `demo/webserver` - LLVM IR HTTP server + WASM fractal client demo.
- `demo/storage` - LLVM IR IPS durability and recovery demo.
- `demo/ui-kit` - Isomorphic UI Kit (LLVM IR to Wasm styling and logic).
- `demo/plaintext` - LLVM IR plaintext HTTP server tailored for the TFB plaintext benchmark.

Each demo has its own `spec` and build pipeline.
